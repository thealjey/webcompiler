cssAutoprefix(data, styleFile, callback)

Adds vendor prefixes to css automatically
Parameters:
Name Type Description
data Object an object containing "code" and "map" strings
styleFile string a full system path to a css file (source and destination files are considered the same, since it's is usually a result of some prior compilation anyway, nothing is ever written to disk)
callback function a callback function, accepts 2 arguments: an array of error messages or null and an object containing "code" and "map" strings
Source:
  • cssAutoprefix.js, line 40
Example
import {cssAutoprefix} from 'webcompiler';

cssAutoprefix({code: 'some css rules', map: 'source map contents'}, '/path/to/a/file.css', function (e, result) {
  if (e) {
    return e.forEach(function (err) {
      console.error(err);
    });
  }
  // result -> {code: string, map: string}
});
