A SASS compiler
Parameters:
Name Type Attributes Default Description
excludeLinter Array.<string> <optional>
 [] names of linters to exclude
importOnceOptions Object <optional>
 {} an object that lets you override default importOnce resolver configuration

 [] an array of additional include paths

import {SASS} from 'webcompiler';

Methods

feDev(inPath, outPath, callback)

Compiles and auto-prefixes an SCSS file for the browser (development mode - faster recompilation).
Parameters:
Name Type Description
inPath string a full system path to the input file
outPath string a full system path to the output file
callback function a callback function, executed after the successful compilation
Example
sass.feDev('/path/to/the/input/file.scss', '/path/to/the/output/file.css', function () {
  // compiled successfully
});

feProd(inPath, outPath, callback, …lintPaths)

Lints, compiles, auto-prefixes, minifies and GZIPs an SCSS file for the browser (production mode).
Parameters:
Name Type Attributes Description
inPath string a full system path to the input file
outPath string a full system path to the output file
callback function a callback function, executed after the successful compilation
lintPaths string <repeatable>
 paths to files/directories to lint (the input file is included automatically)
Example
sass.feProd('/path/to/the/input/file.scss', '/path/to/the/output/file.css', function () {
  // compiled successfully
}, '/lint/this/directory/too');

Performs linting
Parameters:
